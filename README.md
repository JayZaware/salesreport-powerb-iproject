# Retail Sales Dashboard using Power BI

## Project Overview
This project focuses on analyzing retail sales data using Microsoft Power BI. The Superstore dataset was used to build an interactive business intelligence dashboard for analyzing sales, profit, customer behavior, and regional performance.

The dashboard provides visual insights to support business decision-making through interactive charts, KPI cards, and filters.
